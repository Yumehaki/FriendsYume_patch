1.如何安裝 Meisskey

請參閱此處。

2.如何轉成Friends Yume進行介紹（僅限第一次）

首先，如第 1 章介紹 Meisskey。 
 
$ sudo su --misskey # 切換到misskey用戶
$ cd ~/misskey
$ git clone https://github.com/Yumehaki/FriendsYume_patch.git patch
$ cd patch
$ ./update.bash # 如果有任何問題返回，請按 Enter 直到它消失。


3.如何轉成Friends Yume進行介紹（第二次及以後） 

 
$ sudo su --misskey # 切換到misskey用戶
$ cd ~/misskey/patch
$ git pull
$ ./update.bash # 如果您有任何問題，請按 Enter 直到它們消失。


4.如何獲取源代碼 

 
$ cd ~ # 移動到你想放置源代碼的目錄
$ git clone https://github.com/mei23/misskey.git -b 10.102.496-m544
$ cdmisskey
$ git clone https://github.com/Yumehaki/FriendsYume_patch.git -b 10.103.0.0-ypf
$ cd patch
$ ./sourceCheck.bash # 如果您有任何問題，請按 Enter 直到它們消失。
