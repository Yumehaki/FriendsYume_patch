2. How to convert to Friends Yume for introduction (first time only)

First, introduce Meisuki as in Chapter 1. 
 
$ sudo su - misskey #Switch to #misskey user
$ cd ~ /misskey
$ git clone https://github.com/Yumehaki/FriendsYume_patch.git patch
$ cd patch
$ ./update.bash # If any question comes back, press Enter until it disappears.


3. How to convert to Friends Yume for introduction (second and subsequent times) 

 
$ sudo su - misskey # Switch to misskey user
$ cd ~ /misskey /patch
$ git pull
$ ./update.bash # If any question comes back, press Enter until it disappears.


4. How to get the source code 

 
$ cd ~ # Move to the directory where you want to put the source code
$ git clone https://github.com/mei23/misskey.git -b 10.102.496-m544
$ cd misskey
$ git clone https://github.com/Yumehaki/FriendsYume_patch.git -b 10.103.0.0-ypf
$ cd patch
$ ./sourceCheck.bash # If any question comes back, press Enter until it disappears.
