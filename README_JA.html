2.導入しためいすきーにフレンズゆめに変換する方法(初回のみ)

まず、めいすきーを1章のとおりに導入する。 

			$ sudo su - misskey # misskeyのユーザに切り替える
			$ cd ~/misskey
			$ git clone https://github.com/Yumehaki/FriendsYume_patch.git patch
			$ cd patch
			$ ./update.bash #もし何かしらの質問文が帰ってきたら、消えるまでEnterを押す。


3.導入しためいすきーにフレンズゆめに変換する方法(2回目以降) 


			$ sudo su - misskey # misskeyのユーザに切り替える
			$ cd ~/misskey/patch
			$ git pull
			$ ./update.bash # もし何かしらの質問文が帰ってきたら、消えるまでEnterを押す。


4.ソースコードの入手方法 


			$ cd ~ #ソースコードを置きたいディレクトリに移動
			$ git clone https://github.com/mei23/misskey.git -b 10.102.496-m544
			$ cd misskey
			$ git clone https://github.com/Yumehaki/FriendsYume_patch.git -b 10.103.0.0-ypf
			$ cd patch
			$ ./sourceCheck.bash # もし何かしらの質問文が帰ってきたら、消えるまでEnterを押す。
