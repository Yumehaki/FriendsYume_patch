2. Как преобразовать в Friends Yume для ознакомления (только в первый раз)

Сначала представьте Мейсуки, как в главе 1. 
 
$ sudo su - misskey # Переключиться на пользователя misskey
$ cd ~ /misskey
$ git clone https://github.com/Yumehaki/FriendsYume_patch.git patch
$ cd patch
$ ./update.bash # Если возникнут какие-либо вопросы, нажимайте Enter, пока они не исчезнут.


3. Как преобразовать в Friends Yume для ознакомления (второй и последующие разы) 

 
$ sudo su - misskey # Переключиться на пользователя misskey
$ cd ~ /misskey /patch
$ git pull
$ ./update.bash # Если возникнут вопросы, нажимайте Enter, пока они не исчезнут.


4. Как получить исходный код 

 
$ cd ~ # Перейти в каталог, в который вы хотите поместить исходный код
$ git clone https://github.com/mei23/misskey.git -b 10.102.496-m544
$ cd misskey
$ git clone https://github.com/Yumehaki/FriendsYume_patch.git -b 10.103.0.0-ypf
$ cd patch
$ ./sourceCheck.bash # Если у вас возникнут вопросы, нажимайте Enter, пока они не исчезнут.
